Свойства объекта window

Объект Window

Даёт доступ к окну браузера. Обладает следующими свойствами:

  • document
  • history
  • location
  • navigator
  • screen
  • status

document

  • свойство объекта window
  • объект
  • предоставляет доступ к элементам html-страницы

Цепочка свойств

                    var d, i, i_src;
                    d = window.document;
                    i = document.images[0];
                    i_src = i.src;
                
                    var i_src = window.document.images[0].src;

Доступ к элементу страницы

document.getElementById(id)
