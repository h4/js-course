Объект Date

  • Встроенный объект JavaScript
  • Предоставляет инструменты для работы с датой и временем
  • Объекты создаются конструктором new Date()

Создание экземпляра объекта Date

  • var a = new Date();
  • var b = new Date("May 21, 1958 10:15 AM");
  • var c = new Date(1958, 4, 21, 10, 15);

new Date();

new Date("May 21, 1958 10:15 AM");

Wed May 21 1958 10:15:00 GMT+0400 (MSK)

new Date(1958, 4, 21, 10, 15);

Wed May 21 1958 10:15:00 GMT+0400 (MSK)

  • Год, месяц, день, час, минута, день
  • Месяц – начинается с «0»
  • Можно добавить 6-й параметр

Свойства объекта Date()

У объекта нет свойств

Методы объекта Date()

Более 40 свойств

  • getProp() - возвращает значение, не имеет аргументов
    • getFullYear()
    • getMilliseconds()
  • setProp(args) - устанавливает значение, переданное в аргументе
    • Date.setFullYear(year,month,day)
    • setMilliseconds(millisec)
  • Прочие методы

setTimeout(func|code,milliseconds)

Выполняет JS код через указанный промежуток времени

  • Глобальный метод
  • func – имя выполняемой функции
    • function myFunc() {...};
    • var t = setTimeout(myFunc, 1000);
  • code – строка кода
    • var t = setTimeout('alert("hello");', 1000);
  • Обычно создаётся как значение переменной

setInterval(func|code,milliseconds)

Выполняет JS код через равные промежутки времени

Остановка выполнения интервалов

                    var t = setTimeout(func, 1000),
                        i = setInterval(func, 1000);
                    clearInterval(i);
                    clearTimeout(t);
