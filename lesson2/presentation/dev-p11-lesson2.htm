Логическое И (&&)

Возвращает true, если и левое, и правое значения возвращают true

  • var a = 1, b =0; c = a && b; // c = false
  • var a = 0, b =1; c = a && b; // c = false
  • var a = 1, b =1; c = a && b; // c = true
  • var a = 0, b =0; c = a && b; // c = false

Логическое ИЛИ (||)

Возвращает true, если или левое, или правое значения возвращают true

  • var a = 1, b =0; c = a || b; // c = true
  • var a = 0, b =1; c = a || b; // c = true
  • var a = 1, b =1; c = a || b; // c = true
  • var a = 0, b =0; c = a || b; // c = false

Логическое НЕТ (!var)

Возвращает true, если операнд ложен

  • var a = 1; c = !a; // c = false
  • var a = 0; c = !a; // c = true

Циклы

Повторение определённой последовательности действий, до наступления события, определённого условным выражением

  • Цикл do .. while
    • Событие может быть внешним
    • Может не завершиться никогда
  • Цикл for
    • Условие выполнения описыватеся в определении цикла
    • Может не завершиться никогда

Цикл do .. while

                    var x = 1;
                    do {
                      alert(x);
                      x++;
                    }
                    while (x < 5);

Цикл for

                    for (var x = 1; x < 5; x++) {
                      alert(x);
                    }
