Условные операторы и циклы

Выполнение кода в зависимости от выполнения логических условий

Условные операторы и циклы

  • if .. else .. elseif - если
  • do .. while – делать пока...
  • for – делать пока выполняется условие цикла
  • switch – поиск совпадений

Операторы if .. else .. elseif

                    if (condition) {
                        // Блок кода
                        // JS code
                    }

condition — переменная или логическое выражение

Операторы if .. else .. elseif

                    if (condition) {
                        // JS code
                    } else {
                        // JS code
                    }

Операторы if .. else .. elseif

                    if (condition) {
                        // JS code
                    } elseif (condition) {
                        // JS code
                    } else {
                        // JS code
                    }
                
                    if (a == 0) {
                        // JS code
                    } else  if (a == 1) {
                        // JS code
                    } elseif (a == 2) {
                        // JS code
                    } else {
                       //  JS code
                    }

Тернарный оператор «?»

                    if (age > 14) {
                       var access = true;
                    } else {
                        var access = false;
                    }
                
var access = age > 14 ? true : false;

Конструкция switch

  1. Проверка на равенство перенной с набором вариантов значений
  2. Строгая проверка на равенство (===)
  3. Эквивалентная нескольким if подряд
                    switch (a) {
                    case "1":
                      // JS code;
                      [break;]
                    case "2":
                      // JS code;
                      [break;]
                    default:
                      // JS code;
                      [break;]
                    }

Инструкция break

  1. «Выход» из блока кода (switch или цикла)

Циклы

Повторение определённой последовательности действий, до наступления события, определённого условным выражением

Цикл с предусловием

  • Цикл while ..
    • Может не выполниться ни одного раза

Цикл с постусловием

  • Цикл do .. while
    • Событие может быть внешним
    • Выполняется как минимум один раз
    • Может не завершиться никогда

Цикл do .. while

                    var x = 1;
                    do {
                      alert(x);
                      x++;
                    }
                    while (x < 5);

Цикл for. Цикл со счётчиком

  • Опеределения начального значения счётчика
  • Условие выхода из цикла
  • Выражение изменения счётчика
  • Условие выполнения описыватеся в определении цикла
  • Может не завершиться никогда
  • Используется чаще всего

Цикл for

                    for (var x = 1; x < 5; x++) {
                      alert(x);
                    }

Инструкция continue

  1. Переход к следующей итерации цикла

Функции

Функция – последовательность команд, которую можно использовать повторно.

Основные понятия:

  • имя функции
  • параметры
  • тело функции

Мы уже знакомы %)

                    alert(message);
                    confirm(message, default)

Определение функции

                    function speakBot() {
                        var msg;
                        msg = prompt("Скажи что нибудь", "");
                        alert("Ты сказал" + msg);
                    }

Вызов функции

                    speakBot();

Параметры функции

                    function speakBot(name) {
                        var msg;
                        msg = prompt("Скажи что нибудь", "");
                        alert(name + " сказал " + msg);
                    }

Параметры функции

                    var name = prompt("Как тебя зовут?", "Анонимус");
                    speakBot(name);

Параметры функции

  1. Все параметры являются необязательными
  2. Нельзя присвоить начальное значение параметру внутри ()
  3. Если параметр не передан — он считается равным undefined

Возвращаемое значение

                    function addTax(price, tax) {
                        var result = price * (1 + tax / 100);
                        return result;
                    }

Результат функции можно присваивать переменной

                    var fullPrice = addTax(price, tax)

Область видимости переменной

  • Глобальная
    • Переменная доступна для чтения и записи в любой точке сценария
  • Локальная
    • Переменная доступна только внутри функции

Типы функций

  • Именованные
  • Анонимные

Массивы

Стандартный (встроенный) объект JavaScript.

Содержит набор пронумерованных элементов (массив c числовыми индексами)

Примеры массивов

  • Месяцы года
  • Товары в интернет-магазине
  • Все картинки на веб-странице
    • document.images
  • Все теги h2 на веб-странице
    • document.getElementsByTagName("h2")

Способы создания массива

  • Использование конструктора
  • Литеральная запись

Конструктор массива

                    var days = new Array(7);
                    days[0]="Понедельник";
                    days[1]="Вторник";
                    ...
                    days[6]="Воскресенье";
                    var abc = new Array ("a", "b", "c", "d", "e");

Литеральная запись

                    var emptyArr = [];
                    var cars = ["Saab",
                        "Volvo",
                        "Uaz"];
                
                    var dummy = new Array(7);
                    var seven = new Array("7");

Не используйте new Array()
