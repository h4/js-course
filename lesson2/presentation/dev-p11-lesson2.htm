Типы данных

Элементарные (простые) типы данных

  1. string - строка
  2. number - число
  3. boolean - логическое
  4. null - null
  5. undefined - значение отсутствует

Объекты

Объект - основной тип данных в JavaScript.

В JS всё – объект

Примитиыне типы — тоже объекты

  • alert (typeof ("0"));
  • alert (typeof (String("0")));
  • alert (typeof (new String("0")));

Исключения

Undefined – не объект

Условные операторы и циклы

Выполнение кода в зависимости от выполнения логических условий

Условные операторы и циклы

  • if .. else .. elseif - если
  • do .. while – делать пока...
  • for – делать пока выполняется условие цикла
  • switch – поиск совпадений

Операторы if .. else .. elseif

                    if (condition) {
                        // Блок кода
                        // JS code
                    }

condition — переменная или логическое выражение

Операторы if .. else .. elseif

                    if (condition) {
                        // JS code
                    } else {
                        // JS code
                    }

Операторы if .. else .. elseif

                    if (condition) {
                        // JS code
                    } elseif (condition) {
                        // JS code
                    } else {
                        // JS code
                    }
                
                    if (a == 0) {
                        // JS code
                    } else  if (a == 1) {
                        // JS code
                    } elseif (a == 2) {
                        // JS code
                    } else {
                       //  JS code
                    }

Тернарный оператор «?»

                    if (age > 14) {
                       var access = true;
                    } else {
                        var access = false;
                    }
                
var access = age > 14 ? true : false;

Конструкция switch

  1. Проверка на равенство перенной с набором вариантов значений
  2. Строгая проверка на равенство (===)
  3. Эквивалентная нескольким if подряд
                    switch (a) {
                    case "1":
                      // JS code;
                      [break;]
                    case "2":
                      // JS code;
                      [break;]
                    default:
                      // JS code;
                      [break;]
                    }

Инструкция break

  1. «Выход» из блока кода (switch или цикла)

Циклы

Повторение определённой последовательности действий, до наступления события, определённого условным выражением

Цикл с предусловием

  • Цикл while ..
    • Может не выполниться ни одного раза

Цикл с постусловием

  • Цикл do .. while
    • Событие может быть внешним
    • Выполняется как минимум один раз
    • Может не завершиться никогда

Цикл do .. while

                    var x = 1;
                    do {
                      alert(x);
                      x++;
                    }
                    while (x < 5);

Цикл for. Цикл со счётчиком

  • Опеределения начального значения счётчика
  • Условие выхода из цикла
  • Выражение изменения счётчика
  • Условие выполнения описыватеся в определении цикла
  • Может не завершиться никогда
  • Используется чаще всего

Цикл for

                    for (var x = 1; x < 5; x++) {
                      alert(x);
                    }

Инструкция continue

  1. Переход к следующей итерации цикла
