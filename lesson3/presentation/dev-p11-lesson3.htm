Объект

Основной тип данных в JS

Особенности объектов

  • Объект – ассоциативный массив (словарь, хеш)
  • Основа ООП

Создание объекта

  • Оператор new Object
  • Литеральная форма
  • Использование конструктора

Создание объекта

                    var rabbit = new Object();
                    rabbit.legs = 4;
                    rabbit.size = "small";
                    alert(rabbit);
                    console.log(rabbit);

Создание объекта

                    var rabbit = {
                        legs: 4,
                        size: "small"
                    };
                    alert(rabbit);
                    console.log(rabbit);

Типы объектов

  • Встроенные (базовые) объекты
  • Пользовательские объекты

Встроенные объекты

  • Array
  • Boolean
  • Date
  • Error
  • Function

Встроенные объекты

  • Math
  • Number
  • Object
  • RegExp
  • String

Встренные объекты пишутся с заглавной буквы

Обращение к свойствам объекта

                    alert(rabbit.legs);
                    alert(rabbit["legs"]);
                    var attrVal = "size";
                    alert(rabbit[attrVal]);

Конструктор объекта

                    function animal(objName, objLegs, objSize, objRunMethod) {
                        this.name = objName;
                        this.legs = objLegs;
                        this.size = objSize;
                        this.runMethod = objRunMethod;
                        this.run = function() {alert(this.runMethod)};
                    }

Создание объекта из конструктора

                    var rabbit = new animal("rabbit", 4, "small", "jump"),
                        wolf = new animal("wolf", 4, "medium", "walk");
                     
                    rabbit.run();

Оператор this

  • Указывает на текущий объект
  • В глобальной области видимости указывает на глобальный объект

Глобальный объект

  • В браузере – объект window
  • Можно вызывать методы и свойства глобального объекта не указывая на него
                    alert(this);
                    alert("Hello");
                    window.alert("Hello");

Прототип объекта

  • Свойство объекта
  • Свойства прототипа передаются всем экземплярам объекта
  • Одна из ключевых особенностей JS
