Массивы

Стандартный (встроенный) объект JavaScript.

Содержит набор пронумерованных элементов (числовой массив)

Примеры массивов

  • Месяцы года
  • Товары в интернет-магазине
  • Все картинки на веб-странице
    • document.images
  • Все теги h2 на веб-странице
    • document.getElementsByTagName("h2")

Функции

Функция – последовательность команд, которую можно использовать повторно.

Определение функции

                    function speakBot() {
                        var msg;
                        msg = prompt("Скажи что нибудь", "");
                        alert("Ты сказал" + msg);
                    }

Вызов функции

                    speakBot();

Параметры функции

                    function speakBot(name) {
                        var msg;
                        msg = prompt("Скажи что нибудь", "");
                        alert(name + " сказал " + msg);
                    }

Параметры функции

                    var name = prompt("Как тебя зовут?", "Кто-то");
                    speakBot(name);

Возвращаемое значение

                    function addTax(price, tax) {
                        var result = price * (1 + tax / 100);
                        return result;
                    }
                    alert(addTax(235, 12.3));

Создание объекта

  • Оператор new Object
  • Литеральная форма

Создание объекта

                    var rabbit = new Object();
                    rabbit.legs = 4;
                    rabbit.size = "small";
                    alert(rabbit);
                    console.log(rabbit);

Создание объекта

                    var rabbit = {
                        legs: 4,
                        size: "small"
                    };
                    alert(rabbit);
                    console.log(rabbit);

Конструктор объекта

                    function animal(objName, objLegs, objSize, objRunMethod) {
                        this.name = objName;
                        this.legs = objLegs;
                        this.size = objSize;
                        this.runMethod = objRunMethod;
                        this.run = function() {alert(this.runMethod)};
                    }

Создание объекта из конструктора

                    var rabbit = new animal("rabbit", 4, "small", "jump"),
                        wolf = new animal("wolf", 4, "medium", "walk");
                     
                    rabbit.run();

Оператор this

  • Указывает на текущий объект
  • В глобальной области видимости указывает на глобальный объект

Глобальный объект

  • В браузере – объект window
  • Можно вызывать методы и свойства глобального объекта не указывая на него
                    alert(this);
                    alert("Hello");
                    window.alert("Hello");
