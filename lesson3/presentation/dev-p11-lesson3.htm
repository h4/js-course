Объект

Объект

  1. Cоставной (сложный) тип данных
  2. Неупорядоченная коллекция свойств (ассоциативный массив, хеш)
  3. Свойство имеет имя и значение
  4. Значение свойства может быть любого типа
  5. Объекты могут взаимодействовать друг с другом

Атрибуты объекта

  • Свойства
  • Методы

Свойства и методы объекта

  • У встроенных объектов предопределённый набор свойств и методов
  • Свойства и методы можно добавлять, изменять и удалять

Конструктор объекта

                    function Animal(objName, objSize) {
                        this.name = objName;
                        this.legs = 4;
                        this.size = objSize;
                        this.eat = function(obj) {
                            return "I eat " + obj;
                        };
                    }

Создание объекта из конструктора

                    var rabbit = new Animal("rabbit", "small"),
                        wolf = new Animal("wolf", "medium");
                     
                    wolf.eat(rabbit);

Констуктор

  • Функция, возращающая объект
  • Объект возвращается явно (return {}) или при помощи оператора this
  • Имя конструктора принято писать с заглавной буквы

Оператор this

  • Указывает на текущий объект
  • В глобальной области видимости указывает на глобальный объект

Прототип конструктора

                    function Animal(objName, objSize) {
                        this.name = objName;
                        this.legs = 4;
                        this.size = objSize;
                    }
                    Animal.prototype.eat = function(obj) {
                        return "I eat " + obj;
                    };

Прототип конструктора

  1. Свойства прототипа конструктора доступны для всех объектов, созданных этим конструктором
  2. Значения свойств прототипа одинаковы для всех порождённых объектов
  3. Чаще всего в прототипе определяют функции

setTimeout ( func|code, timeout )

  1. Выполняет JS код (func|code) через указанный промежуток времени (timeout)
  2. Промежуток времени задаётся в миллисекундах
  3. Код выполняется один раз

setTimeout ( func|code, milliseconds )

  • Глобальный метод (в браузере)
  • func – имя выполняемой функции
    • function myFunc() {...};
    • var t = setTimeout(myFunc, 1000);
  • code – строка кода
    • var t = setTimeout('alert("hello");', 1000);
  • Обычно создаётся как значение переменной

setInterval ( func|code, milliseconds )

Выполняет JS код через равные промежутки времени

Остановка выполнения интервалов

                    var t = setTimeout(killBill, 1000),
                        bill = setInterval(living, 50);
                    function killBill () {
                        clearInterval(bill);
                        console.log("Bye, Bill");
                    }
                    function living() { console.log("I'm alive"); }
