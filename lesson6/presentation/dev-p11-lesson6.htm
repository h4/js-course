Объект Style

Позволяет изменить css-свйоства элемента

                    document.getElementById("el1").style.background="#EEE";
                    document.getElementById("el1").style.color="";

Изменение css-свойства элемента

  1. Найти нужный элемент
    • document.getElementById
    • document.getElementsByTagName(tagname)[index]
    • document.body
  2. Обратиться к свойству style
  3. Обратиться к нужному свойству объекта style
  4. Изменить значение свойства

Желательно кешировать объект style

Особенности свойств объекта style

  • Тип свойства — строка
  • Отображает результат, а не само свойство
  • background-color → backgroundColor
  • Возвращает значение, указанное в атрибуте style

Отображение свойств

                    <p id="new">AJAX</p>
                    elStyle = document.getElementById('new').style;
                    elStyle.color = "#F00";
                    console.log(elStyle.color);
                    // "rgb(255, 0, 0)"

Наследование свойств

                    p {font-size: 1.2em;}
                    <p id="new">AJAX</p>
                    el = document.getElementById('new');
                    console.log(el.style.fontSize);
                    // ""

Реальное (вычисленное) значение свойства

                    p {font-size: 1.2em;}
  • window.getComputedStyle(elem, null).fontSize
  • elem.getComputedStyle.fontSize //IE < 9

Особенности getComputedStyle | getComputedStyle

  • Доступно только для чтения
  • Отображает результат вычислений
  • В Firefox пиксели могут быть дробными
  • Свойство currentStyle сохраняет единицы измерения
  • background-color → backgroundColor

Кроссбраузерный способ

                    function computedStyle(el) {
                        if (window.getComputedStyle) {
                            return getComputedStyle(elem, null);
                        } else {
                            return = elem.currentStyle;
                        }
                    }

style.cssText

                    el.style.cssText="color: red !important; \
                        background-color: yellow; \
                        width: 100px; \
                        text-align: center; \
                        -ufo-diameter: 5; \
                        ";
