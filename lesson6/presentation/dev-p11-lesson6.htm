Объект style

Объект style

Даёт доступ к изменению визуального отображения элемента

  • document.getElementById("el1").style.background="#EEE";

Изменение свойств элемента

  • Изменение свойств объекта style
  • Изменение класса элемента
  • Подключение дополнительной таблицы стилей

Изменение стиля элемента

  1. Найти нужный элемент
    1. document.getElementById
    2. document.getElementsByTagName(t)[num]
    3. document.body
  2. Обратиться к объекту-свойству style
  3. Обратиться к нужному свойству объекта style
  4. Изменить значение свойства

Особенности свойств объекта style

  • Тип свойста — строка
  • background-color => backgroundColor
  • Возвращает значение, указанное в атрибуте style

Желательно кешировать объект style

Реальное (вычисленное) значение свойства

p {font-size: 1.2em;}
  • elem.currentStyle.fontSize
  • window.getComputedStyle(elem, null).fontSize

Недостатки el.style

  • Много рутинной работы

Изменение имени класса

Если нужно одновременно измненить несколько свойств

                    var el = document.getElementById("elID");
                    el.className = "newClassName";
